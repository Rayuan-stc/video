FH8862 是新一代支持 4K 实时轻智能、双目广角 IP Camera SoC 芯片。芯片集成了新一代高性能 ISP 图像处理模块和 Smart H.265 视频压缩编码模块，集成了人工智能处理引擎，具备优异的图像处理能力、极高的编码质量和智能处理能力。芯片支持双路 MIPI CSI-2 CMOS Sensor 输入，支持 2 路 MIC 音频输入，支持外接 32bit 或 16bit DDR3/3L 存储颗粒。 芯片拥有完备的应用所需外设接口，包括 USB2.0、SDIO 和 Ethernet PHY 等，可适用多种网络摄像机产品。

* 处理器：Operating Frequency : 900MHz ；32KB I-CACHE, 32KB D-CACHE Per core ；128KB L2 Cache per Core ；
* 控制器 ：Operating Frequency : 594MHz ；16KB I-CACHE, 16KB D-CACHE
* 复位：复位管理模块对整个芯片、各功能模块的复位进行统一的管理。上电复位的管理和控制；WatchDog 复位管理和控制；系统软复位、功能模块单独软复位控制
* 时钟输入：时钟要求 24MHz 晶体或晶振；精度要求±20ppm
* 实时时钟：实时时钟 RTC（Real Time Clock）用于实现时间显示和定时报警功能。支持 2.0V～3.3V电池供电
* 温度传感器：芯片内部集成温度传感器（T-Sensor），温度检测范围为-40～125℃。
* I2C 接口：芯片有 I2C0、I2C1、I2C2、I2C3 和 I2C4 五组 I2C 控制器，其中 I2C2 为 Slave，其余为 Master。五组 I2C 中 I2C0、I2C3 和 I2C4 固定为 1.8V 电压域。支持多主设备时的总线仲裁；支持标准地址（7-bit）和扩展地址（10-bit）；传输速率支持标准模式（100Kbit/s）和快速模式（400Kbit/s）
* SPI 接口：芯片有 SPI0、SPI1 和 SPI2 三组 SPI 控制器，其中 SPI0 和 SPI1 为 Master，SPI2 为 Slave， SPI2 与 SPI1 管脚复用。接口时钟频率可编程，最高频率 120MHz；串行数据帧长度 8bit；支持 DMA 操作 l 支持 SPI 同步串行接口，支持单帧和连续帧格式；支持 SPI 全双工工作模式，相位可配置
* GPIO 接口：每个 GPIO 管脚可以配置为输入或输出。这些管脚用于生成特定应用的输出信号或采集特定应用的输入信号。GPIO 可以根据电平或跳变值产生可屏蔽的中断。作为输入管脚时，可作为中断源，中断信号电平有效或沿有效可配置；作为输出管脚时，每个 GPIO 管脚都可以独立地清 0 或置 1；支持 Debounce 去抖动功能
* PWM 输出：芯片的 PWM（Pulse Width Modulation）可以提供脉宽调制信号输出，芯片共有 14 通道 PWM 输出。支持占空比可配置；支持通道延迟输出；支持通道间相位控制
* 网络接口：支持单网口（内置 EPHY 或外接 EPHY 二选一）；内置 10/100M Ethernet PHY；支持 RGMII 或者 RMII Ethernet MAC 输出
* SDIO ：SDIO 用于对 SD 卡、eMMC 器件的读写等操作，并通过 SDIO 协议实现对扩展外设（如 蓝牙、WiFi 等）的支持。芯片提供 SDIO0 和 SDIO1 两组独立的 SDIO 控制器，其中 SDIO1 支持 1.8V 或 3.3V eMMC 启动，SDIO0 只支持 3.3V eMMC 启动。支持 1bit/4bit 模式；支持 eMMC 4.41；兼容 SD2.0，支持 SDHC，用于本地缓存；兼容 SDIO 2.0/1.10，用于无线扩展（WiFi）；最高时钟频率 50MHz
* USB2.0：芯片支持1个USB2.0接口，支持工作在 USB2.0 Host 模式或 USB2.0 Device 模式，不支持动态切换。USB2.0 Host 支持 USB2.0 协议规定的 480Mbps/12Mbps/1.5Mbps 三种传输速率，支持控制传输、批量传输、同步传输、中断传输四种基本的传输方式。USB2.0 Device 支持 480Mbps/12Mbps 两种传输速率，支持控制传输、批量传输、 同步传输、中断传输四种基本的传输方式。
